群晖docker搭建 koush/scrypted 连接摄像头

2021-10-14 / 2 min read

源代码地址：https://github.com/koush/scrypted

  1. 群晖docker安装 sudo docker pull koush/scrypted
  2. sudo docker run --network host -v $(pwd)/scrypted:/server/volume koush/scrypted
  3. 设置时区 TZ=Asia/Shanghai



    打开网址：Scrypted 现在运行在：https://localhost:10443/
  4. 插件（plugin）--安装（Install Plugin）
  5. 安装 HomeKit
  6. onvif
  7. rtsp
  8. Rebroadcast Plugin
  9. OpenCV
  10. 如果不想用rtsp 直接安装onvif即可
  11. 添加摄像头后设置参考
  12. RTSP海康威视摄像头设置参考： rtsp://admin:12345@192.0.0.64:554/h265/ch1/main/av_stream
  13. 主码流取流:
    rtsp://admin:12345@192.0.0.64:554/h265/ch1/main/av_stream
    子码流取流:
    rtsp://admin:12345@192.0.0.64:554/h265/ch1/sub/av_stream
    如果摄像机密码是a12345678,IP是192.168.1.,64，RTSP端口默认554未做改动，是H.264编码，那么
    rtsp://10.1.1.xx:554/h264/ch1/sub/av_stream
    【如果是H.265编码的，那么将H.264替换成H.265即可】
    参考：海康威视rtsp地址



  14. 设置好后在手机中添加HomeKit 配件

添加至手机就可以正常浏览摄像头画面了

海康威视摄像头支持双向对讲功能若无法双向对讲，需要设置 视音频
将音频，音频编码设置为：G.711ulaw ，设置完成后摄像头即可在HomeKit进行双向对讲。
