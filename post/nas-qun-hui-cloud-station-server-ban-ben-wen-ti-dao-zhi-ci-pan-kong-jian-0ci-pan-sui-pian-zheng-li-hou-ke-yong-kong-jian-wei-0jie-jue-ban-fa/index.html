Nas 群晖 （Cloud Station Server版本问题导致磁盘空间0）（磁盘碎片整理后可用空间为0）解决办法

2021-08-15 / 1 min read

遇到磁盘空间为0解决办法。

方法1：（cloud软件处理方法）
1.关闭全部cloud版本。
2.控制面板---删除 Cloud Station Server（新版本cloud统一归纳到（Drive）），删除时不要保留cloud的版本数据。（实测有用）2021年08月15日

方法2：（ 命令整理文件夹）
1.控制面板打开 ssh功能， 进入群晖登录。

  1. cd /volume(你储存目录)/
  2. du * -sh (查看文件夹空间占用情况：)
  3. 文件系统重新排列： nohup btrfs fi defrag -y /volumeX/shared folder -r &
  4. 文件系统重新排列需要时间请耐心等待。正常要5-12小时。（未测试）
