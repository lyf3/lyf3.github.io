群晖 docker 安装 heimdall快捷方式网页 安装教程。

2021-08-19 / 1 min read

heimdall快捷方式安装教程。

命令安装教程：
1、ssh打开群晖。

2、sudo-i 获取最高权限。

3、下载 heimdall容器。

4、命令： docker pull linuxserver/heimdall

5、在群晖docker文件夹下新建heimdall文件夹储存heimdall文件。

6、下载完成后使用安装命令。

docker run -d \
--name=linuxserver-heimdall1 \
-e TZ="Asia/Shanghai" \
-e PUID=“1000” \
-e PGID=“1000” \
-v /volume1/docker/heimdall:/config \
-p 90:80 \
-p 1443:443 \
--restart unless-stopped \
linuxserver/heimdall:latest

7、完成测试网页。

WEB安装教程：
1、WEB页面docker--注册表--搜索 heimdall， 安装linuxserver/heimdall 映像。

2、映像启动linuxserver/heimdall 。

3.设置 linuxserver/heimdall 容器。

4.设置容器信息。端口设置，地区设置。


5、完成测试网页。

应用地址: https://registry.hub.docker.com/r/linuxserver/heimdall/
应用推荐下载 https://apps.heimdall.site/search?search=Synology
